# Invoice System Enhancement — Software Engineering Simulation

This project was developed as part of the New York Jobs CEO Council Software Engineering Job Simulation with Forage.  
The simulation focused on identifying and resolving bugs in an automated billing system and adding new feature enhancements.

### Key Features
- Corrected discount-application logic to ensure discounts are applied before tax.
- Added comment-tracking functionality to invoices to support clearer communication between sender and recipient.
- Improved code readability and maintainability.

### Skills Demonstrated
Python • OOP • Debugging • Clean Code • Feature Implementation
